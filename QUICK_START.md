# Inicio Rápido - Project Manager

Inicia en 5 minutos.

## Instalación

```bash
python main.py
```

## Primer Proyecto (3 minutos)

1. **Crear Usuario**: Menú → Gestionar Usuarios → Crear nuevo usuario
2. **Seleccionar Usuario**: Menú → Gestionar Usuarios → Seleccionar usuario
3. **Crear Proyecto**: Menú → Gestionar Proyectos → Crear nuevo proyecto
4. **Trabajar**: Menú → Trabajar con Proyecto Actual

## Operaciones Comunes

- **Ver tablero**: Visualizar todas las columnas y tareas
- **Agregar tarea**: Crear nueva tarea con prioridad
- **Buscar tarea**: Encontrar por título o descripción
- **Ver estadísticas**: Métricas del proyecto

## Datos

Los datos se guardan automáticamente en: data/projects.json

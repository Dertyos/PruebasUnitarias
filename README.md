# Project Manager

Un gestor profesional de proyectos y tareas estilo Trello desarrollado en Python con arquitectura orientada a objetos, persistencia de datos y interfaz interactiva de línea de comandos.

## Características

- **Gestión Completa de Usuarios**: Crear, listar, seleccionar y eliminar usuarios
- **Gestión de Proyectos**: Crear proyectos con columnas personalizables
- **Gestión de Tareas**: Crear, editar, buscar y mover tareas entre columnas
- **Sistema de Prioridades**: Asignar prioridades (Baja, Media, Alta, Urgente)
- **Estados de Tareas**: Pendiente, En Progreso, Completada, Bloqueada
- **Asignación de Tareas**: Asignar tareas a usuarios
- **Etiquetado**: Agregar etiquetas personalizadas a las tareas
- **Estadísticas**: Ver estadísticas del proyecto por estado, prioridad y asignación
- **Persistencia de Datos**: Guardar datos en JSON para recuperación persistente
- **CLI Interactiva**: Interfaz amigable con menús navegables

## Instalación

```bash
# Requisitos: Python 3.8+
python main.py
```

## Uso

1. Crear usuario en "Gestionar Usuarios"
2. Seleccionar usuario actual
3. Crear proyecto en "Gestionar Proyectos"
4. Trabajar con tareas en "Trabajar con Proyecto Actual"

Ver QUICK_START.md para más información.
